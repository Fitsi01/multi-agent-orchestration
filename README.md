# multi-agent-orchestration
В этом репозитории вы найдете мульти агентную систму для написания контент-плана на неделю, который включает 2 поста в телеграм, 3 сценария для видео (лонг, шортс, тикток), ИИ-обложка для видео и статья на заданную тематику.
ENG: in this repository features a multi-agent system for creating a weekly content plan, which includes two Telegram posts, three video scripts(YT long, shorts and TikTok), an AI video preview, and an article on a given topic.
