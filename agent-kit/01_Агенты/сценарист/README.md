# СЦЕНАРИСТ (маркетинговое агентство)

## Назначение

Сценарии для видеороликов агентства: YouTube (5–15 мин), Shorts (30–60 сек), Reels (15–30 сек). Темы — маркетинг, продажи, кейсы, обучение клиентов.

## Команда

`@сценарист`

## Как использовать

```
@сценарист напиши сценарий на тему [тема]
@сценарист YouTube, 10 минут, тема: как читать отчёт Директа
@сценарист Shorts про три ошибки в UTM-разметке
```

## Что на выходе

Файл в `02_Контент/сценарии/` с именем `ГГГГ-ММ-ДД_тема-kebab-case.md` (латиница/цифры/дефисы в slug, см. скилл).  
Внутри — таблица сцен: тайминг / речь / визуал / звук.
