# ПРЕВЬЮШНИК (маркетинговое агентство)

## Назначение

Генерирует обложки для видео по **теме выпуска** из контент-плана или по явному запросу (кейс, разбор, обучающий ролик агентства).

## Команда

`@превьюшник`

## Что создаёт

- AI-картинка (PNG) через GenerateImage
- Текстовое ТЗ для дизайнера (заголовок на превью, стиль, цвета)

## Как использовать

`@превьюшник сделай превью для темы [название]`  
`@превьюшник обложка под ролик про [канал/метрику]`
