# Research-агент: зарубежный маркетинг для агентства

## Что делает

По запросу обходит зарубежные источники (Reddit, YouTube, Twitter/X, TikTok), собирает **маркетинговые темы и кейсы** (рост, реклама, контент, аналитика), проверяет применимость для работы в РФ и сохраняет черновой отчёт для ручного отбора.

Имя выходного файла **`ГГГГ-НН-схемы.md`** — историческое имя для совместимости с планировщиком и скиллами; содержимое — маркетинговые темы («карточки»), не «схемы заработка».

**Вызов:** «ресёрч», «найди темы», «обход источников» или `@ресёрч`.

## Выходной файл

`02_Контент/ресёрч/ГГГГ-НН-схемы.md`

Один файл на неделю. Повторный запуск за ту же неделю **дополняет** существующий файл.

## ROI

| Было | Стало |
|------|-------|
| 3–5 ч ручного мониторинга трендов | 20 мин на разбор готового списка карточек |

## Ограничения

- Финальное решение «брать тему или нет» — за тобой. Агент ставит вердикт-рекомендацию.
- Twitter/X и TikTok — в основном через web_search. Часть материалов агент не достанет — это норма.
- Reddit JSON API работает без авторизации, но может быть замедлен.

## Расписание запуска

Cursor не имеет встроенного планировщика. Три варианта:

### Вариант A — Ручной запуск (рекомендуется на старте)

Написать в чат: `@ресёрч` или «запусти обход источников».  
Например, 2 раза в неделю: понедельник и четверг.

### Вариант B — Windows Task Scheduler

Запускает Cursor с нужным workspace по расписанию (см. прежнюю версию README в истории git при необходимости). Полная автоматизация без участия пользователя может требовать Background Agent или внешний триггер.

### Вариант C — Напоминание

Telegram-бот или календарь: «Пора обновить ресёрч недели».
