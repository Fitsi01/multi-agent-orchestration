# Шаблон записи одной темы (карточка для ресёрча агентства)

Агент копирует этот блок для каждой найденной темы и заполняет все поля.  
Незаполненные поля не оставлять пустыми — писать «не указано» или «не найдено».  
Заголовок блока в отчёте может оставаться **«Схема №N»** для совместимости со старыми файлами — по смыслу это **тема / кейс**.

---

```markdown
## Схема №N — [Название темы одной строкой]

| Поле | Содержание |
|------|------------|
| Суть | Что делают: какой маркетинговый результат, для какого типа бизнеса |
| Канал / инструмент | Площадки, стек, martech (рекламный кабинет, CRM, аналитика, AI-инструменты) |
| Где применить | Канал продвижения или воронка (лидоген, ecom, B2B-кабинеты и т.п.) |
| Заявленный KPI / ориентир | Цифры из источника: ROAS, CTR, бюджет, срок до результата — брать как есть, не «доказывать» |
| Барьер внедрения | Низкий / Средний / Высокий (время команды, бюджет, компетенции) |
| Риски | Политика площадок, сезонность, конкуренция, качество данных, правовые ограничения |

### Применимость в РФ

- [ ] Площадки и инструменты доступны в РФ без VPN или есть рабочий аналог
  - Аналог если нет: _(VK Реклама / Директ / Roistat / Yandex Metrica / другое)_
- [ ] Оплата медиа и сервисов реализуема для российского клиента
  - Альтернатива: _(российский кабинет, локальный подрядчик, другое)_
- [ ] Нет критических правовых рисков
  - Флаги: _(ФЗ «О рекламе» — недостоверные гарантии / 152-ФЗ — персданные / другое)_
- [ ] Российская SMB-аудитория готова платить за такой результат
  - Оценка спроса: _(есть кейсы в РФ / ниша сырая / другое)_

**Вердикт:** ✅ Брать / ⚠️ Адаптировать / ❌ Пропустить

**Обоснование вердикта:** _(1–2 предложения почему)_

**Источник:** [заголовок или описание](URL)  
**Дата материала:** ДД.ММ.ГГГГ (приблизительно)
```

## Пример заполненной записи

```markdown
## Схема №1 — Воронка лидогена на Webinar + Meta Ads (EU кейс)

| Поле | Содержание |
|------|------------|
| Суть | Лид-магнит → серия писем → вебинар → оффер услуги; масштабирование через lookalike |
| Канал / инструмент | Meta Ads, Zoom/WebinarJam, HubSpot или аналог CRM |
| Где применить | B2B-услуги, консалтинг, агентства |
| Заявленный KPI / ориентир | CPL снижен на 35% за 6 недель теста (по посту) |
| Барьер внедрения | Средний — нужен креатив, посадочная, интеграция форм |
| Риски | Качество базы lookalike, усталость аудитории от вебинаров |

### Применимость в РФ

- [x] Площадки и инструменты доступны в РФ без VPN или есть рабочий аналог
  - Аналог если нет: _VK Реклама + локальный вебинар (TrueConf / Zoom где доступен)_
- [x] Оплата медиа и сервисов реализуема для российского клиента
  - Альтернатива: _VK Реклама, Директ; CRM — Bitrix24 / amoCRM_
- [x] Нет критических правовых рисков
  - Флаги: _нельзя гарантировать CPL без дисклеймера_
- [x] Российская SMB-аудитория готова платить за такой результат
  - Оценка спроса: _много B2B на вебинарах и лид-магнитах_

**Вердикт:** ⚠️ Адаптировать

**Обоснование вердикта:** Логика воронки переносима; стек и юридические формулировки нужно локализовать под РФ и дисклеймеры в рекламе.

**Источник:** [Пример: пост r/marketing о вебинарной воронке](https://reddit.com/r/marketing/...)  
**Дата материала:** апрель 2026
```
