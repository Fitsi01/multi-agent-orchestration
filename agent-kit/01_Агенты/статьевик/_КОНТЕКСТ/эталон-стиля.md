# Эталон стиля — статьевик (маркетинговое агентство)

Ориентиры стиля для статей в закрытый Telegram-клуб **клиентов и подписчиков агентства**.

## Эталонная статья

**Файл:** `01_Агенты/статьевик/_КОНТЕКСТ/примеры/пример-статья-кейс-агентства.md` (входит в этот репозиторий)  
**Тема:** кейс агентства — снижение стоимости заявки, честные цифры и шаги  
**Почему эталон:** задаёт тон: практика, ограничения, структура из 8 частей без воды

## Ключевые принципы стиля

### Структура
- 8 именованных Частей — не просто заголовки
- Каждая часть самодостаточна: читатель должен понять суть, даже если читает только её
- Объём: 2 000–4 000 слов

### Голос
- Обращение на «ты» — всегда
- Разговорный язык: как объясняешь клиенту за кофе
- Никаких клише: «уникальный», «революционный», «невероятный»
- Никаких абстракций: конкретные числа, шаги, названия площадок и инструментов

### Примеры правильных формулировок

| Плохо | Хорошо |
|-------|--------|
| «Вы можете значительно улучшить показатели» | «CTR вырос с 1,2% до 3,8% после смены креатива» |
| «Найди клиентов в интернете» | «Собери 20 компаний из 2ГИС по запросу [ниша] [город] — им пиши вот этим шаблоном» |
| «Инструмент недоступен в РФ» | «Прямой доступ закрыт. Рабочий обход: [аналог] + что теряешь по функциям» |
| «Существуют определённые риски» | «Риск: разгон бюджета без отсева ботов. Как обойти: чек-лист из пяти пунктов в Части 7» |

### Шаблоны и промпты
- В fenced code block — всегда
- Переменные в квадратных скобках: `[ниша]`, `[город]`, `[продукт клиента]`
- После блока — одно предложение: что менять под клиента агентства

### Финал статьи
- Часть 8 заканчивается одним конкретным действием прямо сейчас
- Не «подумай о стратегии» — а «открой отчёт за вчера и проверь три метрики из списка»
- Последняя строка — дисклеймер (не юридическая/финансовая консультация, результат зависит от ниши и исполнения)

## Эталонные примеры внешние

- Материалы в духе «гайд для практиков» — разговорный стиль, структура по частям, без пустого мотивашна
