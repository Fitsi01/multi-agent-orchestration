# СТАТЬЕВИК (маркетинговое агентство)

## Назначение

Пишет полные статьи для **закрытого Telegram-клуба клиентов** агентства. Тема — из контент-плана недели или по явному запросу. Статья — 2 000–4 000 слов, самодостаточная, структура 8 частей.

## Команда

`@статьевик`

## Как использовать

```
@статьевик напиши статью по теме недели
@статьевик статья по теме: ретаргетинг через CRM
@статьевик выпуск в клуб — тема из ресёрча №3
```

## Что на выходе

Файл в `02_Контент/статьи/` с именем `ГГГГ-ММ-ДД_суть-темы.md`.  
Внутри — 8 именованных частей: контекст → механика → инструменты → шаги → первые результаты → промпты и шаблоны → риски → итог.
