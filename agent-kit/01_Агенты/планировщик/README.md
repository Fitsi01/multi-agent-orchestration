# Планировщик контента (маркетинговое агентство)

## Что делает

Читает готовый ресёрч-файл недели, отбирает темы с вердиктами ✅/⚠️ и строит контент-план:  
какая **тема недели** идёт в закрытый клуб клиентов, какие ролики снимать и когда, что постить в открытый TG агентства.

**Вызов:** «планировщик», «составь план», «контент-план» или `@планировщик`.

## Выходной файл

`02_Контент/планы/ГГГГ-НН-план.md`

Один файл на неделю. Повторный запуск за ту же неделю перезаписывает файл.

## ROI

| Было | Стало |
|------|-------|
| 30–60 мин на раздумья «что публиковать» | 5 мин на одобрение готового плана |
| Ролики и посты без системы | Расписание: кто, что, когда, какой агент |

## Место в пайплайне

```
@ресёрч → 02_Контент/ресёрч/ГГГГ-НН-схемы.md → @планировщик → 02_Контент/планы/ГГГГ-НН-план.md → @статьевик / @сценарист / @копирайтер / @превьюшник (или один вызов @пакет)
```

## Ограничения

- Финальный выбор темы недели — за тобой. Агент предлагает, не решает.
- Полный текст выпуска для клуба агент не пишет — только ставит в план (его пишет @статьевик или вручную).
- Если ресёрч-файл за текущую неделю не создан — сначала запусти `@ресёрч`.

## Связанные агенты

| Агент | Роль в пайплайне |
|-------|-----------------|
| `@ресёрч` | Создаёт файл тем недели, который читает планировщик |
| `@статьевик` | Полный выпуск в закрытый клуб по теме недели из плана |
| `@сценарист` | Скрипты для TikTok/YouTube по ролям из плана |
| `@копирайтер` | Посты в открытый TG по плану |
| `@превьюшник` | Обложки под ролики по теме недели |
