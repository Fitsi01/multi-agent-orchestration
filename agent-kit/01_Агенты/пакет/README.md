# ПАКЕТ (маркетинговое агентство)

## Назначение

Оркестратор полного контент-пакета. Один запуск — **8 файлов** по одной одобренной **теме из ресёрча** (в т.ч. «тема недели» из плана).

## Команда

`@пакет`

## Что создаёт за один запуск

1. Статья для закрытого TG-клуба клиентов (2 000–4 000 слов)
2. Сценарий YouTube-лонг (5–10 мин)
3. Сценарий YouTube Shorts (60 сек)
4. Сценарий TikTok (60 сек)
5. AI-обложка PNG
6. Текстовое ТЗ к обложке
7. Тизер в открытый TG-канал агентства
8. Анонс в закрытый TG-клуб

## Как использовать

`@пакет тема №3 — Performance на Ozon`  
`@пакет тема недели`  
`@пакет с планом` / `@пакет с нуля` / `@пакет от ресёрча`
